## Installation

```bash
brew install pastesheet
```

## Usage

| Action | Shortcut |
| :--- | :--- |
| **Toggle App** | `Cmd` + `Shift` + `V` |
| **Navigate** | `↑` `↓` `←` `→` Arrow Keys |
| **Paste / Select** | `Enter` |
| **Edit Item** | `Cmd` + `E` |
| **Close** | `Esc` |

**Pro Tip:** Move your mouse cursor to the **right edge** of the screen to quickly peek at your clipboard!
