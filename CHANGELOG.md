# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-03

### Added
- **Official Beta Release** 
- Multi-folder organization system for clipboard history.
- Performance-driven core built with Rust & Tauri.
- macOS native support (dmg and Homebrew distribution).
- **Pro Feature:** "Edge Peek" - quick clipboard access by moving mouse to the right screen edge.
- Search functionality with intuitive keyboard navigation.
- Customizable settings (Auto-hide toggle, etc.).
- Markdown-based item editing support.

### Changed
- Improved README with Beta status and Windows roadmap.
- Optimized keyboard shortcuts for better UX (`Cmd+E` to edit, etc.).

### Fixed
- Fixed directory navigation bugs when using arrow keys.
- Resolved focus issues in search and modal inputs.
- Fixed UI jittering during list navigation.
- Corrected external monitor paste behavior.

## [Unreleased]
- Windows platform support.
- Sync functionality across devices.
